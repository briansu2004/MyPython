Installation

Linux:
Download x86/x64: https://nodejs.org/en/download
Notes: Extract to folder, add bin folder to path

Mac:
Download .pkg: https://nodejs.org/en/download
Notes: Open downloaded file, drag file into system

Windows:
Download .msi: https://nodejs.org/en/download
Notes: Run downloaded file, follow instructions on screen


===


Running Programs

Linux/Mac/Windows:
Run file: Open terminal, navigate to folder containing file, do 'node file.js'
Use interpreter: Open terminal, run node
